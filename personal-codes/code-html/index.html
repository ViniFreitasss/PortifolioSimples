Mémoria RAM

As mémorias RAM's também conhecidas como ramdom acess memory, são dispositivos de armazenamento de informações volateis, isso significa que as informações contidas nos mesmos são apagadas após o desligamento da máquina

  • SDRAM

  • DDR

  • DDR2

  • DDR3

  • DDR4

Mémoria ROM

As mémorias ROM's também conhecidas como read only memory ou mémoria de somente leitura são dispotivos que armazenam informações constantes ou seja informações que são mantidas até mesmo após o desligamento da máquina, apesar do seu nome dizer "read only" (somente leitura) é possivel apagar e até mesmo reescrever o conteudo de alguns modelos

  • ROM

  • MASK-ROM

  • PROM

  • EPROM

  • EEPROM
